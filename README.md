# RunTogether

###### 建大约跑

1. 项目简介：该项目为new bee团队在大二暑假参加山东省大学生软件设计大赛提交的作品，获得三等奖。
2. 应用技术:Bmob后端云+百度地图sdk+百度鹰眼轨迹。
3. 主要功能:记步，约跑，附近的人以及发送动态。
